# vgc_limitless_webscraping
Web scrapper para escanear e povoar banco de dados dos recentes campeonatos de pokémon vgc.

Não mais limitado propositalmente à somente a primeira página de resultados de torneios recentes da Regulation M-A de Pokémon Champions, agora buscando todas as páginas de torneio do regulamento.

Futuramente será feito o mesmo para o regulamento M-B, e talvez com outros sites de torneio se a necessidade surgir.
